# Privacy Policy for MiMo Mobile

## Data Collection
MiMo Mobile does NOT collect any personal data. All communication happens locally between your phone and your PC.

## Data Storage
- Server settings (IP address, port) are stored locally on your Android device using DataStore
- No data is sent to external servers
- No analytics or tracking

## Network Communication
- All communication uses WebSocket on your local network
- No data leaves your WiFi network
- PIN authentication prevents unauthorized access

## Open Source
MiMo Mobile is open source under the MIT License. You can review the complete source code at any time.
