MiMo Mobile

## Short Description (80 chars)
Remote control for MiMo Code CLI from your Android phone

## Full Description (4000 chars)
MiMo Mobile is the ultimate companion app for MiMo Code CLI users. Control your AI coding assistant from anywhere in your home or office using just your Android phone.

**Features:**
• **Real-time Chat** — Send prompts to MiMo Code and watch responses stream in live. No more switching between terminal and browser.
• **Build Monitor** — Beautiful animated visualization of your project build pipeline. See files, stages, and progress in real-time.
• **Remote Desktop** — View your PC screen directly on your phone. Perfect for monitoring long-running tasks.
• **Terminal Access** — Execute any shell command remotely. Full terminal power in your pocket.
• **File Browser** — Navigate your project files, read code, and make quick edits from your phone.
• **Device Manager** — Control connected Android devices via ADB. Take screenshots, install apps, send input commands.

**How it works:**
1. Start the MiMo Mobile server on your PC (one command)
2. Install the app on your Android phone
3. Enter your PC's IP address and connect
4. That's it! Start controlling MiMo Code from your phone

**Requirements:**
• Python 3.13+ on your PC
• MiMo Code CLI installed
• Android 8.0+ (API 26)
• Both devices on the same WiFi network

**Privacy:**
• All communication stays on your local network
• No data sent to external servers
• PIN-based authentication
• Open source (MIT License)
